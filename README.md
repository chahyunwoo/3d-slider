# 3D-Slide (By React TypeScript)
3D 입체 슬라이더로, 드래그시 카드 회전 및 시야 각도 변경이 가능합니다.

## FEATURES
🟢 React Typescript로 제작

🟢 외부 라이브러리 없이 자체 구현

🟢 드래그 방향에 따라서 자동으로 애니메이션 방향 변경
